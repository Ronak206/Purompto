# Purompto - AI Prompt Engineering SaaS

An AI-powered prompt generation platform that helps users create perfect prompts through intelligent clarifying questions. Built with modern technologies for a professional, ChatGPT-like experience.

![TypeScript](https://img.shields.io/badge/TypeScript-5.9-blue?style=flat-square&logo=typescript)
![MongoDB](https://img.shields.io/badge/MongoDB-Atlas-green?style=flat-square&logo=mongodb)

## ✨ Features

### 🤖 AI-Powered Prompt Engineering
- **Intelligent Questions** - AI asks clarifying questions to understand your exact needs
- **Streaming Responses** - Watch prompts generate in real-time with SSE
- **Perfect Results** - Get optimized prompts that work first time
- **Smart Retry** - Retry goes through analysis to check if enough info before regenerating

### 💬 ChatGPT-like Experience
- **Real-time Chat History** - Every message saved instantly to MongoDB
- **Session Recovery** - Resume any conversation from sidebar
- **Grouped History** - Chats organized by Today, Yesterday, This Week
- **Collapsible Sidebar** - Maximize your workspace
- **Concurrent Chat Support** - Multiple chats can generate simultaneously
- **AbortController** - Stop generation when switching chats, with resume option

### 🎨 Modern UI/UX
- **Dark/Light Theme** - Toggle between themes
- **Responsive Design** - Works on desktop and mobile
- **Auto-expanding Input** - Textarea grows with content (52px → 200px)
- **Send Button** - Bottom-right corner, like z.ai style
- **Smooth Animations** - Professional feel with loading states
- **Copy to Clipboard** - One-click copy for prompts
- **User/AI Avatars** - Flex layout with avatars beside message bubbles
- **ChatGPT-style Bubbles** - 18px radius, proper spacing, professional look
- **shadcn/ui Components** - Avatar, Badge, Button, Input, ScrollArea, Sheet, Sidebar, etc.

### 🎯 Generated Prompt Styling
- **Teal/Cyan Gradient** - Beautiful gradient background for generated prompts
- **Tips Section** - Helpful tips shown between conversation and result
- **Summary Display** - Shows prompt summary in the result header
- **Markdown Rendering** - Full markdown support in messages

### 👨‍💼 Admin Panel
- **User Management** - Create, update, delete users
- **Access Control** - Enable/disable user accounts
- **Notes System** - Track payment status, notes per user
- **Chat Storage** - View all chat messages per user
- **Prompt History** - View all prompts generated by users

## 🚀 Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/Ronak206/Purompto.git
cd Purompto
npm install
```

### 2. Environment Setup
Create `.env` file:
```env
# Database
MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/purompto

# Authentication
JWT_SECRET=your-random-jwt-secret-min-32-chars
ADMIN_SECRET=your-admin-panel-password

# AI API
OPENROUTER_API_KEY=your-openrouter-api-key
OPENROUTER_API_KEY_2=backup-api-key-optional
OPENROUTER_API_KEY_1=another-backup-optional

# App URL
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Access the App
- **Home**: http://localhost:3000
- **Admin Panel**: http://localhost:3000/admin

## 🗄️ Database Models

### User
```typescript
{
  email: string;              // Unique email
  password: string;           // Hashed with bcrypt
  name?: string;              // Display name
  isActive: boolean;          // Account status
  totalPromptsGenerated: number;
  notes?: string;             // Admin notes
  createdAt: Date;
  updatedAt: Date;
}
```

### Chat
```typescript
{
  chatId: string;             // Unique chat ID
  userId: ObjectId;           // Owner reference
  title: string;              // Chat title
  messages: [{                // Conversation history
    id: string;
    role: 'user' | 'assistant';
    content: string;
    questions?: string[];
    questionReasons?: string[];
    createdAt: Date;
  }];
  status: 'active' | 'completed' | 'generated' | 'remaining';
  result?: string;            // Final generated prompt
  summary?: string;           // Prompt summary
  createdAt: Date;
  updatedAt: Date;
}
```

## 🔌 API Reference

### Authentication
```http
POST /api/auth/login      # Login
POST /api/auth/logout     # Logout
POST /api/auth/signup     # Create account (admin only)
GET  /api/auth/me         # Current user
GET  /api/auth/token      # Get JWT token
```

### Chat Operations
```http
GET  /api/chat            # List all chats
GET  /api/chat?chatId=x   # Get specific chat
POST /api/chat            # Create/update chat
DELETE /api/chat?chatId=x # Delete chat
```

### AI Operations
```http
POST /api/analyze         # Generate questions (checks if ready)
POST /api/clarify         # Get clarifications
POST /api/generate        # Generate prompt (streaming SSE)
```

### Admin Operations
```http

GET  /api/admin/chats           # List all chats
```

### Other

## 🎯 Business Model

Purompto is designed as a **managed SaaS service**:

1. **Admin creates accounts** for clients after receiving payment
2. **Clients login** and use the service - no payment processing on platform
3. **Admin manages access** - can enable/disable accounts

### Workflow
```
Client pays → Admin creates account → Client uses service
     ↑                                        │
     └──────── Admin controls access ←────────┘
```

## 💡 How It Works

### Prompt Generation Flow
```
1. User describes task → AI asks clarifying questions
2. User answers questions → AI gathers more details if needed
3. AI confirms "ready to generate" → Streaming prompt generation
4. Result displayed with tips → Copy or continue refining
```

### System Prompt Behavior
- AI MUST ask clarifying questions first
- Gathers: purpose, audience, tone, format, length, context
- Only generates when `readyToGenerate: true`
- Retry also goes through analysis to verify readiness
